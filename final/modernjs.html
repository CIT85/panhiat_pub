What is ES?

ES stands for ECMAScript, scripting language specification for JavaScript, a programming language, is an implementation of ECMAScript specifications

ES6 stands for ECMAScript 6 which was publish in 2015 also known as ECMAScript 2015 (ES6) which inroduced new features to JavaScript.

Destructuring Assignment in ES6

  1. Array Destructuring:

    Unpack values from arrays into distinct variables.

  2. Object Destructuring:

    Extract values from objects into variables with the same property names.

  3. Default Values:

    Set default values during the destructuring process.

  4. Rest Syntax:

    Gather remaining elements into a new array using the rest syntax.

  5. Nested Destructuring:

    Destructure nested structures, such as objects inside arrays or vice versa.

  6. Parameter Destructuring:

    Apply destructuring directly to function parameters for cleaner function signatures.

  7. Alias:

    Assign the extracted values to variables with different names.

Two ES6 Features

Arrow Functions and Template Literals

Arrow Functions

Details

Arrow functions provide a concise syntax for writing functions. They have a shorter syntax compared to traditional function expressions and share the same lexical `this` as their surrounding code.

Example:

                                
                                    const add = (a, b) => a + b;

                                    console.log(add(2, 3)); // Outputs 5

Template Literals

Details

Template literals are string literals allowing embedded expressions. They make it easier to create dynamic strings with variables or expressions by using backticks (`) instead of quotes.

Example:

                                
                                    const name = 'World';

            
                                    const greeting = `Hello, ${name}!`;
                                    console.log(greeting); // Outputs "Hello, World!"

Table of ES6 Features

Eight ES6 Features

Feature Description
let and const Block-scoped variable declarations. `let` allows reassignment, while `const` is for constants.
Arrow Functions Concise syntax for writing functions, with an implicit return for one-liners.
Template Literals String literals with embedded expressions using backticks (`) for easier string formatting.
Spread and Rest Operators Spread operator (...) for array and object manipulation, while rest operator (...) gathers elements into an array.
Destructuring Assignment Extract values from arrays or properties from objects into distinct variables.
Classes Object-oriented programming with class syntax for constructor functions and inheritance.
Modules ES6 introduced a native module system to organize and structure code.
