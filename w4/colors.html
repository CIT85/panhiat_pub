My Rainbow Box Model

The box model is essential to an element. It allows developers to modify the content, padding, border and margin as desire. In the photo above, I set the content of the box model to 200 pixel. I prefer to use pixel for the content on this element because it is static and does not change base on viewport. In my color palette, I made the border double for the style and the width 1em relative to the parent element of div. I also added 10% margin on each div element to make space around the element yet making it dynamic and adjustible to the viewport.
